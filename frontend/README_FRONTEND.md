Frontend (React + Vite)
----------------------
Como rodar (Windows / VS Code):

1. Abra o terminal integrado no VS Code.
2. Vá para a pasta frontend:
   cd frontend
3. Instale dependências (apenas na primeira vez):
   npm install
4. Rode o frontend:
   npm run dev
5. Abra no navegador:
   http://localhost:5173
